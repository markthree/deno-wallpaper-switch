# deno-wallpaper-switch

由 [Deno](https://deno.land/) 实现的自动切换壁纸，可以结合系统的任务计划实现每日自动切换壁纸。

<br />

## Usage

### start

```shell
deno task start
```

### compile

```shell
deno task compile
```
